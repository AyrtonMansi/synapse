# Forum Templates

Templates for common discussion types on the Synapse community forum.

## Bug Report Template

```markdown
## Bug Description
<!-- Clear description of the bug -->

## Steps to Reproduce
1. Step one
2. Step two
3. Step three

## Expected Behavior
<!-- What should happen -->

## Actual Behavior
<!-- What actually happens -->

## Environment
- OS: [e.g., Ubuntu 22.04]
- Node Version: [e.g., 1.2.3]
- GPU: [e.g., RTX 4090]
- Network: [Mainnet/Testnet]

## Logs
```
<!-- Paste relevant logs here -->
```

## Additional Context
<!-- Screenshots, videos, etc. -->
```
