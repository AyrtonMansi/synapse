Quick Start

Get up and running with Synapse AI in minutes.

1. Get an API Key

Sign up for a Synapse AI account and generate an API key from the dashboard.

2. Install the SDK

npm install @synapse-ai/sdk
pip install synapse-ai-sdk

3. Make Your First Request

import { SynapseSDK } from '@synapse-ai/sdk';

const sdk = new SynapseSDK({ apiKey: 'your-api-key' });

const response = await sdk.chatCompletion({
  model: 'gpt-4',
  messages: [{ role: 'user', content: 'Hello!' }]
});

console.log(response.choices[0].message.content);
import asyncio
from synapse_sdk import SynapseSDK, CompletionRequest, Message, MessageRole

async def main():
    sdk = SynapseSDK(api_key='your-api-key')
    
    response = await sdk.chat_completion(
        CompletionRequest(
            model='gpt-4',
            messages=[Message(role=MessageRole.USER, content='Hello!')]
        )
    )
    
    print(response.choices[0].message.content)
    await sdk.close()

asyncio.run(main())

Authentication

All API requests require authentication using an API key passed in the Authorization header.

Authorization: Bearer YOUR_API_KEY
Keep your API keys secure

Never expose your API keys in client-side code or public repositories.

Streaming

For real-time applications, use streaming to receive partial responses as they're generated.

JavaScript

const stream = sdk.streamChatCompletion({
  model: 'gpt-4',
  messages: [{ role: 'user', content: 'Tell me a story' }]
});

for await (const chunk of stream) {
  const content = chunk.choices[0]?.delta?.content;
  if (content) {
    process.stdout.write(content);
  }
}

Python

async for chunk in sdk.stream_chat_completion(request):
    content = chunk.choices[0].delta.get("content", "")
    print(content, end="", flush=True)
