## ✨ Key Features

### For AI Developers
- **Permissionless Access** - No KYC, no approval process
- **Pay-per-Compute** - Only pay for what you use
- **Streaming Payments** - Real-time payment for ongoing jobs
- **Dispute Resolution** - Fair arbitration for compute disputes

### For Node Operators
- **Passive Income** - Earn HSK tokens for providing compute
- **Reputation System** - Build trust and earn higher rewards
- **Flexible Scheduling** - Run jobs on your terms
- **ZK Verification** - Cryptographic proof of work

### For Token Holders
- **DAO Governance** - Vote on protocol changes
- **Staking Rewards** - Stake HSK for additional yield
- **Fee Sharing** - Earn from network fees

## 📚 Getting Started

### For Developers
```bash
npm install @synapse-network/sdk
```

Visit [docs.synapse.network](https://docs.synapse.network) for API reference and tutorials.

### For Node Operators
```bash
docker run -d --gpus all \
  -e WALLET_ADDRESS=0x... \
  synapse/node:latest
```
